# Classification project
This project aims to predict the type of garment to which a picture belongs.
In this case, we are given ten types of garments: pants, shorts, skirts, sneakers, T-shirts, Dresses...

We start with a normalisation step. The images are modeled by (28,28,3) matrixes. 
The neural network model is trained on a first batch and then tested 
Basicly, we feed in the model with a picture of a trouser. Then we expect the model to tell that it is a trouser and not a skirt, nor a T-shirt nor a dress.


# Projet de Classification

Ce projet vise à prédire le type de vêtement auquel appartient une image.
Dans ce cas, nous avons dix types de vêtements : pantalons, shorts, jupes, baskets, t-shirts, robes...

Nous commençons par une étape de normalisation. Les images sont modélisées par des matrices (28,28,3). 
Le modèle de réseau neuronal est entraîné sur un premier lot, puis testé 
En gros, nous alimentons le modèle avec une image d'un pantalon. Nous attendons ensuite du modèle qu'il nous indique qu'il s'agit d'un pantalon et non d'une jupe, d'un t-shirt ou d'une robe.
